# Dataset Card — Fake and Real News Dataset

## 1. Источник

Kaggle Dataset:  
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Используется для обучения моделей классификации новостей по достоверности.

---

## 2. Структура данных

Файлы:
- Fake.csv — только фейковые новости  
- True.csv — только реальные  

Поля:
- title  
- text  
- subject  
- date  

Размер: около 44 000 записей.

---

## 3. Лицензия

Kaggle Dataset License — свободно для образовательных и исследовательских целей.

---

## 4. Разделение данных

Использовались доли:

- 80% — train  
- 10% — validation  
- 10% — test  

Разбиение фиксировано и детерминировано с константным seed.
